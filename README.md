# 63 Agency - Next.js Frontend

مشروع Next.js مع TypeScript و Tailwind CSS

## التثبيت

قم بتشغيل الأوامر التالية:

```bash
# تثبيت TypeScript والأنواع
npm install -D typescript @types/react @types/node @types/react-dom

# تثبيت Tailwind CSS
npm install -D tailwindcss postcss autoprefixer

# تثبيت ESLint
npm install -D eslint eslint-config-next
```

## التشغيل

```bash
# وضع التطوير
npm run dev

# بناء المشروع
npm run build

# تشغيل الإنتاج
npm start

# فحص الكود
npm run lint
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.
